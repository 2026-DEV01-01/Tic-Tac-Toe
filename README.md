# Tic-Tac-Toe Stateless REST API

### About
Tic-tac-toe (Xs and Os) is a 2-player game. Players alternate marking spaces; X goes first. To win, place three marks in a horizontal, vertical, or diagonal row. If all squares fill without this, the game is a draw. Players cannot reuse marked spaces.
